# Neon Breaker - 霓虹打磚塊

一個使用 React, TypeScript 和 HTML5 Canvas 打造的精細打磚塊遊戲。

## 特色

- **精細動態**: 使用粒子系統與物理碰撞
- **霓虹風格**: 現代感十足的視覺效果
- **關卡系統**: 多樣化的磚塊佈局
- **音效回饋**: 即時的遊戲音效

## 本地開發

### 前置需求

- Node.js (建議 v18 以上)

### 安裝與執行

1. 安裝依賴:
   ```bash
   npm install
   ```

2. 啟動開發伺服器:
   ```bash
   npm run dev
   ```

3. 打開瀏覽器訪問 `http://localhost:3000`

## 技術棧

- **框架**: React 19
- **語言**: TypeScript
- **動畫**: Motion (Framer Motion)
- **樣式**: Tailwind CSS 4
- **圖標**: Lucide React
- **構建工具**: Vite
